## 🐛 常见问题

### 问题 1: 页面空白
**解决方案**: 检查终端是否有错误信息，确保 Nuxt 服务正常运行。

### 问题 2: 图片不显示
**解决方案**: 检查网络连接，图片来自 Unsplash CDN。

### 问题 3: 样式不正确
**解决方案**: 确保 Tailwind CSS 正确配置，品牌色已在 `tailwind.config.ts` 中定义。

### 问题 4: 点击无反应
**解决方案**: 打开浏览器控制台查看是否有 JavaScript 错误。
