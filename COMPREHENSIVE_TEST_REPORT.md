## ⚠️ JavaScript 警告和错误

### 发现的问题

#### 1. Pre-transform Error (非阻塞)

**错误信息**:
```
ERROR Pre-transform error: Failed to resolve import "#app-manifest"
from "node_modules/nuxt/dist/app/composables/manifest.js"
```

**出现次数**: 多次
**严重程度**: 🟡 低 (不影响功能)
**影响**: 仅在开发环境出现，不影响页面加载和功能
**原因**: Nuxt 框架内部的开发环境警告
**建议**: 可以忽略，或等待 Nuxt 框架更新

---

#### 2. Vue 组件警告

**警告信息**:
```
WARN [Vue warn]: Failed to resolve component: ChevronDown
WARN [Vue warn]: Component <Anonymous> is missing template or render function
```

**严重程度**: 🟡 低
**影响**: 不影响页面显示和功能
**可能原因**:
- 某些组件未正确导入
- 动态组件加载问题

**建议**: 检查以下文件中的 ChevronDown 组件导入：
```bash
grep -r "ChevronDown" --include="*.vue" --include="*.ts"
```

---

#### 3. API 超时警告

**警告信息**:
```
WARN [updateInfo] 上游请求失败: https://test.chetuoche.net
[TimeoutError]: The operation was aborted due to timeout
```

**严重程度**: 🟡 低
**影响**: 不影响核心功能
**原因**: 外部 API 超时
**建议**:
- 增加超时时间
- 添加错误处理和降级方案
- 检查外部 API 可用性
